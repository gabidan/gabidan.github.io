Deep learning and Neural Networks

Source: https://www.youtube.com/watch?v=DBVLcgq2Eg0&t=477s

Neural Networks:

  • Image Understanding (semantic segmentation - NN captures some details of the image)

  • Machine Translation

  • Smart Apps (image recognition using a camera, for example)

  • Robotics (NN network is learning a task from a video)

  • Synthesizing/generating faces

  • Image transformation (NN doesn’t know what is an object a or b, it learns both after getting it as an input and performs a transformation)

All powered and trained using backpropagation algorithm. Underlying operation: convolution (give 1d,2d,3d signal, convol and get output operator)

Multiple input channels chain into multiple output channels

Pulling: take a window and do different operations such as take max, ave, or take matrix and do matrix ops

Reduction operations

Chain all these operators together - this is called deep learning

Recurrent NN where output going to itself as input

Deep Learning are trained with Gradient Decsent

Gradient Descent

Compute gradient of one node with respect to the other. Stochastic gradient descent: update weight so slightly that you get an output that you actually want. Update weight with derivative of output with respect to my weight. Intuitivelly, if I give an input, get an output. If output is wrong, compute loss fucntion. Compute the distance between desired output and actual output. Take derivatve with respect to that loss, and update weight according to that, so it gets you closer to the desired output. Do it iteritivately, you eventually get the output you want.
